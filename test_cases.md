# SRM 系统测试用例文档 (v2.0)

## 文档说明

本文档包含 SRM 供应商关系管理系统的完整测试用例，覆盖所有新增功能模块（认证、文件管理、日历）以及原有功能。

**测试环境:**
- 前端: React 19.2.0 + TypeScript 5.9.3 + Vite 7.2.4
- 后端: Supabase (PostgreSQL) + Supabase Auth + Supabase Storage
- 基础 URL: `https://tdrbuwshyebmjqljggww.supabase.co`
